## 幻灯片 2：项目背景与痛点
### 传统学习系统的问题
- ❌ **功能单一**：只能提供单一学习服务
- ❌ **缺乏个性化**：无法根据学生特点定制方案
- ❌ **AI 幻觉**：大模型生成内容可能存在错误
- ❌ **学习路径混乱**：学生不知道学习顺序

### 我们的解决方案
- ✅ **多智能体协同**：14 个 Agent 覆盖学习全流程
- ✅ **个性化学习**：基于学生画像的动态方案
- ✅ **防幻觉机制**：6 维度检测 + 交叉验证
- ✅ **知识图谱**：自动构建最优学习路径

## 幻灯片 5：核心技术 - 防幻觉机制
### 6 维度检测
1. **事实性检测**：关键词匹配 + 交叉验证
2. **逻辑性检测**：前后矛盾检测
3. **一致性检测**：多 Agent 互相校验
4. **时效性检测**：时间敏感词检测
5. **安全性检测**：敏感词过滤
6. **完整性检测**：重复内容检测

### 交叉验证流程
```
AI 生成内容 → 6 维度检测 → 交叉验证 → 通过？ → 输出结果
                                      ↓ 否
                                   标记警告
```

## 幻灯片 7：核心技术 - 多智能体协同
### 消息总线设计
```python
class MessageBus:
    def publish(self, agent_name, message):
        self.messages.append({...})
    
    def subscribe(self, agent_name):
        return [m for m in self.messages if m["from"] == agent_name]
```

### Agent 协同关系
- ProfileAgent → TrackAgent → StrategyAgent → ProfileAgent（闭环）
- TutorAgent → AntiHallucination → MessageBus

## PPT 制作建议

### 设计风格
- **主色调**：蓝色系（#3B82F6）+ 紫色系（#8B5CF6）
- **字体**：微软雅黑 / 思源黑体
- **背景**：浅色渐变或纯色
- **图标**：使用简洁的线性图标

### 排版原则
- **每页不超过 7 行**：保持简洁
- **图文并茂**：多用图表、少用文字
- **重点突出**：使用颜色或加粗标注重点
- **动画适度**：使用简单淡入效果

### 演示技巧
- **提前排练**：至少 3 遍，控制时间
- **语速适中**：每分钟 150-180 字
- **眼神交流**：不要只读 PPT
